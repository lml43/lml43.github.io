Reinforcement Learning for video games

This project is my Master thesis. The purpose of this master thesis is to research on an improvement of the branching architecture for Deep Reinforcement Learning, as an effort to cover more sophisticated games, where the action space may include both continuous and discrete actions (as known as a hybrid action space).

The games with different difficulty levels are built in Unity Editor and the algorithms implementation is powered by Unity ML-Agents, a Python API allows developer to interact and extract necessary information for the training process.

Technology

The project is implemented in Python and is capable to be used with a custom training environment built from Unity Editor, powered by Unity ML-Agents Toolkit.

Human Detection

To fulfill our project, we use HOG algorithm with SVM (Support Vector Machine) to build classification model and pyramid moving – window detector to detect human in images.


Technology

We used Histogram of Gradient (HOG) to extract the feature vectors of images, then classify them by a trained Support Vector Machine (SVM) model. Finally, we need a thing called pyramid moving (a sliding window runs on an image pyramid), which allow us to find out exactly where the objects are located in our image. ​

Open source on Github
